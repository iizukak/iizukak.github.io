CHIP-8

いろいろと FPGA でプロセッサを実装することについて調べていると、CHIP-8 というものにいきついた。英語圏ではゲームボーイやファミコンなどのエミュレータを実装する前段階として、CHIP-8 を実装することが多いようだ。

CHIP-8 は、8 ビットマシンで、命令長は 2 Byte 固定、命令数も 35 と小さく実装しやすい。Verilog や SystemVerilog での実装も Github 上にいくつか確認できた。どうやら SystemVerilog で CHIP-8 を実装することを大学の課題として出しているところがあるらしい。

CHIP-8 は実際の計算機ではなく、インタプリタ言語として開発されたようだ。
